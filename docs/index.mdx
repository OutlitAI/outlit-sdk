## Choose Your Integration

<Tabs>
  <Tab title="Script Tag" icon="code">
    The fastest way to start. Add a single script tag to your website.
    
    ```html
    <script
      src="https://cdn.outlit.ai/stable/outlit.js"
      data-public-key="pk_your_key"
      async
    ></script>
    ```
    
    <Card title="Script Tag Guide" icon="arrow-right" href="/tracking/browser/script">
      Learn more about script tag integration
    </Card>
  </Tab>
  <Tab title="React" icon="react">
    First-class React support with hooks and context.
    
    ```tsx
    import { OutlitProvider } from '@outlit/browser/react'

    export default function App({ children }) {
      return (
        <OutlitProvider publicKey="pk_your_key" trackPageviews>
          {children}
        </OutlitProvider>
      )
    }
    ```
    
    <Card title="React Integration" icon="arrow-right" href="/tracking/browser/react">
      Learn more about React integration
    </Card>
  </Tab>
  <Tab title="Node.js" icon="node-js">
    Server-side tracking for authenticated users.
    
    ```typescript
    import { Outlit } from '@outlit/browser-node'

    const outlit = new Outlit({ publicKey: 'pk_your_key' })

    outlit.track({
      email: 'user@example.com',
      eventName: 'subscription_created',
      properties: { plan: 'pro' }
    })
    ```
    
    <Card title="Node.js SDK" icon="arrow-right" href="/tracking/server/nodejs">
      Learn more about server-side tracking
    </Card>
  </Tab>
</Tabs>

## How It Works

<Steps>
  <Step title="Install the Tracker">
    Add our lightweight tracker to your website or app using a script tag, npm package, or React provider.
  </Step>
  <Step title="Track Events">
    Automatically capture pageviews, form submissions, and engagement time. Track custom events that matter to your business.
  </Step>
  <Step title="Identify Visitors">
    When visitors sign up or log in, call `identify()` or `setUser()` to connect their anonymous history to their profile.
  </Step>
  <Step title="Track the Journey">
    Mark key milestones with `activate()`, `engaged()`, `paid()`, and `churned()` to track the full customer lifecycle.
  </Step>
</Steps>
