## Prerequisites

- An Outlit account with an organization
- An AI tool that supports MCP (Claude Code, Claude Desktop, Cursor, etc.)

## Step 1: Get Your API Key

<Steps>
  <Step title="Open MCP settings">
    Go to [Settings > MCP Integration](https://app.outlit.ai/workspace-profile/mcp) in the Outlit dashboard.
  </Step>
  <Step title="Generate an API key">
    Click **Generate API Key** and give it a descriptive name (e.g., "Claude Code - local dev").
  </Step>
  <Step title="Copy the key">
    Copy the key immediately. It won't be shown again. Store it somewhere safe.
  </Step>
</Steps>

<Warning>
  Treat your MCP API key like a password. It grants read access to your customer data. You can revoke keys at any time from the MCP Integration settings page.
</Warning>

## Step 2: Connect Your AI Tool

<Tabs>
  <Tab title="Claude Code">
    Run this command in your terminal:

    ```bash
    claude mcp add outlit-mcp https://mcp.outlit.ai/sse -- \
      --header "Authorization: Bearer YOUR_API_KEY"
    ```

    Replace `YOUR_API_KEY` with the key you copied in Step 1.
  </Tab>

  <Tab title="Claude Desktop">
    Add the following to your Claude Desktop config file:

    **macOS:** `~/Library/Application Support/Claude/claude_desktop_config.json`
    **Windows:** `%APPDATA%\Claude\claude_desktop_config.json`

    ```json
    {
      "mcpServers": {
        "outlit-mcp": {
          "url": "https://mcp.outlit.ai/sse",
          "headers": {
            "Authorization": "Bearer YOUR_API_KEY"
          }
        }
      }
    }
    ```

    Replace `YOUR_API_KEY` with the key you copied in Step 1. Restart Claude Desktop after saving.
  </Tab>

  <Tab title="Cursor">
    Add the following to `.cursor/mcp.json` in your project root (or your global Cursor config):

    ```json
    {
      "mcpServers": {
        "outlit-mcp": {
          "url": "https://mcp.outlit.ai/sse",
          "headers": {
            "Authorization": "Bearer YOUR_API_KEY"
          }
        }
      }
    }
    ```

    Replace `YOUR_API_KEY` with the key you copied in Step 1.
  </Tab>
</Tabs>

## Step 3: Verify the Connection

Once connected, ask your AI tool to call the `outlit_schema` tool:

```
Show me the available Outlit analytics tables.
```

You should see a list of tables including `events`, `customer_dimensions`, `user_dimensions`, and `mrr_snapshots`. If you get an authentication error, double-check your API key.

## Server URLs

| Environment | URL |
|-------------|-----|
| Production  | `https://mcp.outlit.ai` |
| Staging     | `https://staging.mcp.outlit.ai` |
| Local dev   | `http://localhost:3001` |

<Note>
  Most users should use the production URL. Staging and local dev URLs are for Outlit contributors and development purposes.
</Note>

## Testing with cURL

You can verify your API key works before configuring an AI tool:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://mcp.outlit.ai/sse
```

A successful connection returns an SSE stream. Press `Ctrl+C` to stop.

## Managing API Keys

From [Settings > MCP Integration](https://app.outlit.ai/workspace-profile/mcp) you can:

- **Generate** new API keys with descriptive names
- **Copy** existing keys
- **Revoke** keys that are no longer needed

Revoking a key immediately disconnects any AI tools using it.
