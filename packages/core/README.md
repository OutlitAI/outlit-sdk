# @outlit/core

Shared types and utilities for the Outlit browser and Node.js tracking SDKs.

Most applications should install `@outlit/browser` or `@outlit/node` directly. Use this package when building lower-level integrations on top of Outlit's shared tracking types and client utilities.

Outlit is the real-time understanding of every customer, the infrastructure agents use to automate customer operations.
