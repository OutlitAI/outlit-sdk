# @outlit/browser

Browser tracking SDK for Outlit customer context, with React and Vue bindings.

Use this package to send page views, product events, identity, activation, form activity, and customer billing signals from client-side applications into Outlit.

Outlit is the real-time understanding of every customer, the infrastructure agents use to automate customer operations.

## Install

```bash
npm install @outlit/browser
```
